Sobre JavaScript

A linguagem JavaScript inicialmente foi criada rodar no navegador, mas com a evolução da tecnologia, graças ao motor V8 desenvolvido pela Google para rodar JavaScript no Google Chrome foi possível retirar o JavaScript do contexto de um navegador e rodar códigos fora desse ambiente, dando origem a várias tecnologias, como Node.js, React, React Native, etc.

Segue um exemplo de código JavaScript (uma função simples para definir se um número é par ou impar): 

						
function isEven(number) {
	return number % 2 == 0;
}

isEven(7);
